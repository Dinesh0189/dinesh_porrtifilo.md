# Student Portfolio

Add your short portfolio as a Markdown file and open a Pull Request.

## What you need to do
1. Create a new branch named **your-name** (e.g., `priya-sharma`).
2. In the `students/` folder, add a file named **your-name.md** (e.g., `students/priya-sharma.md`) using the template below.
3. Commit, push your branch, and open a Pull Request to `main`.
